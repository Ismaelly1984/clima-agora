Sobre o projeto e o autor

O Clima Agora é um projeto criado para oferecer uma experiência de clima rápida, segura e intuitiva, explorando o poder do web design moderno e das Progressive Web Apps. O foco é apresentar previsões de 3 horas e de 5 dias, com ênfase em probabilidade e quantidade de chuva, além de informações essenciais como temperatura, umidade e vento.

Tecnologia

O app utiliza HTML, CSS e JavaScript puro, sprite SVG para ícones, CSS crítico para melhorar o primeiro paint e Service Worker com navegação preload e cache do shell. As chamadas de API são feitas por um proxy serverless, que mantém a API key longe do navegador.

Autor

Ismael Nunes é o desenvolvedor do Clima Agora. Apaixonado por UX, performance e segurança, atua na construção de experiências web eficientes e acessíveis. Você pode saber mais no perfil: https://ismaelly1984.github.io/.
